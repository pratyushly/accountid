# accountid

First release for open use!

The package that you have been looking to create [URL friendly short IDs just like YouTube](https://www.youtube.com/watch?v=gocwRvLhDf8)

## Installation

Using npm:
```shell
$ npm i accountid
```

Using yarn:
```shell
$ yarn add accountid
```

Using pnpm:
```shell
$ pnpm i accountid
```
